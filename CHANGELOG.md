# 更新日志

## [1.1.0] - 2026-01-30

### 新增
- **Prompt 管理系统**：任务驱动的 Prompt 配置 (`config/prompts.yaml`)
  - 支持 5 种任务：翻译摘要、重要性排序、分类、关键词提取、情感分析
  - 支持模型特定配置（Qwen、GLM 等）
  - 支持变量替换
- **智能批处理**：动态批次大小，根据内容长度自动调整
- **失败重试机制**：批处理失败时自动降级为逐条处理
- **测试结果目录**：`tests/results/` 用于保存测试结果（不提交到 git）

### 优化
- **RSS 采集器**：添加超时设置，避免采集卡住
- **GLM 模型配置**：修复智谱 API 调用方式（使用 OpenAI 兼容模式）
- **Prompt 完整性**：所有任务 Prompt 添加"必须处理每一条"的要求
- **模型切换脚本**：更新 GLM 配置为正确的 API 地址

### 修复
- 修复 GLM-4-Flash 模型返回空响应的问题
- 修复批处理时部分数据未被处理的问题
